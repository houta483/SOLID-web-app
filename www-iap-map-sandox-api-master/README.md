# Middleware API project utilizing NEST.js framework
This project will utilize both the Persistence and domain projects.


## Running the app

```bash
#Link Domain Project
$ NPM LINK iap-domain

#Link Persistence Project
$ NPM LINK iap-persistence

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
